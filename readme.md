# DATA.SUBHELION.COM

## Requirements

- Bash
- AWS CLI
- XXHSUM (Optional)

## Installation

with system

```
clone subhelion/data.subhelion.com
edit  subhelion/data.subhelion.com
```

without system

```
git clone git@github.com:subhelion/data.subhelion.com.git
cd data.subhelion.com
script/start
```

import access keys into aws and name the profile "data.subhelion.com"

```
aws configure
```
